Introduction to GenAI and Simple LLM Inference on CPU and Finetuning of LLM Model to Create a Custom Chatbot

Overview

This project aims to introduce beginners to Generative Artificial Intelligence (GenAI) by performing simple Large Language Model (LLM) inference on a CPU and fine-tuning an LLM model to create a custom chatbot. The focus is on providing a hands-on experience with Intel AI tools, particularly for those with an interest in AI and machine learning.

Problem Statement

Participants will learn the basics of GenAI, execute LLM inference on a CPU, and explore the process of fine-tuning a pre-trained LLM model to create a custom chatbot. This involves the following challenges:

  • Pre-trained Language Models: Handling large file sizes that require significant storage and memory.
  • LLM Inference on CPU: Executing inferences using available hardware.
  • Fine-Tuning LLMs: Customizing pre-trained models for specific applications.
  • Creating Custom Chatbots: Using fine-tuned models to develop chatbots with Intel AI tools.

Objectives

  • Gain foundational understanding of Generative AI and its applications.
  • Perform simple LLM inference on a CPU.
  • Understand and execute the fine-tuning process for LLMs.
  • Create a custom chatbot using fine-tuned LLM models and Intel AI tools.

Prerequisites

  • Understanding of Machine Learning concepts.
  • Programming skills in Python.
  • Experience with NLP libraries (e.g., Hugging Face, Transformers).
  • Familiarity with text-based AI models.
